# Prompt 11: Browser Automation with Playwright

Create a robust, production-ready Python automation script using Playwright that:

1. Launches a headless Chromium browser
2. Navigates to a public demo site (e.g. https://demoqa.com or https://the-internet.herokuapp.com)
3. Performs several actions (click, fill form, extract data)
4. Takes full-page and element screenshots
5. Extracts structured data (headings, links, table rows) into clean JSON
6. Saves all artifacts to `./automation-results/`
7. Includes comprehensive logging, error handling, and retry logic
8. Supports both headed and headless modes via CLI argument

Also provide:
- `requirements.txt`
- Instructions to install Playwright browsers
- Example command to run the script
- How to view results

Make the script safe and respectful (no aggressive scraping).
