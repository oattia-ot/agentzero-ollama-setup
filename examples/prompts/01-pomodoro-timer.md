# Prompt 1: Premium Pomodoro Timer

Create a premium, responsive single-file Pomodoro timer using Tailwind CSS and vanilla JavaScript. Include:

- Circular SVG progress indicator
- Work (25 min) / Short break (5 min) / Long break (15 min) modes
- Web Audio API sound notifications
- Keyboard shortcuts (Space = start/pause, R = reset)
- Dark + light theme toggle with localStorage
- Session counter and statistics
- Clean, modern, professional UI suitable for productivity apps

Output the complete self-contained HTML file.
